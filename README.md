# Python Number Guessing Game

A simple command-line number guessing game written in Python while practicing core programming concepts.

## Features
- Random number between 1 and 100
- Player receives "Too high" or "Too low" hints
- Guess counter showing current attempt
- Maximum number of guesses allowed
- Option to replay the game

## Concepts Practiced
- Functions
- While loops
- Conditional logic (if / elif / else)
- User input handling
- Random number generation
- Loop control with break
- Counters and state tracking
